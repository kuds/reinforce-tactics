# Maps

Reinforce Tactics features a variety of maps designed for different game modes and player counts. Each map offers unique strategic challenges through terrain layout, building placement, and chokepoints.

## Map Legend

- **Green** - Plains (walkable terrain)
- **Dark Green** - Forest (provides defense bonus)
- **Blue** - Water/Ocean (impassable)
- **Gray** - Mountains (impassable, provides attack bonus when adjacent)
- **Brown** - Roads (faster movement)
- **Red** - Player 1 structures
- **Blue Tint** - Player 2 structures
- **Green Tint** - Player 3 structures
- **White/Light Gray** - Neutral Towers/Buildings

## Creating Custom Maps

Want to create your own maps? Use the built-in map editor:

```bash
python3 main.py --mode map_editor
```

See the [Map Editor Guide](https://github.com/kuds/reinforce-tactics/blob/main/MAP_EDITOR.md) for detailed instructions.

### Map Requirements

- Minimum size: 6x6 tiles
- Each player must have exactly one Headquarters (HQ)
- All players should have roughly equal access to neutral structures
- Consider balance when placing terrain features

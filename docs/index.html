Game Features

A comprehensive tactical combat system with diverse units, structures, and strategic depth

⚔️

3 Unit Types

Warriors (melee), Mages (ranged + paralysis), and Clerics (healing + support) with unique abilities and combat mechanics including counter-attacks and status effects.

🏰

Structure Capture

Control Towers, Buildings, and Headquarters. Each structure generates income and serves strategic purposes. Capturing enemy HQ wins the game!

💰

Economic System

Manage gold income from controlled structures, purchase units strategically, and balance offensive and defensive investments.

🎮

Multiple Game Modes

Human vs Human, Human vs Bot, or pure AI training. Includes save/load functionality and replay system for game analysis.

🤖

AI Opponents

Built-in SimpleBot provides challenging baseline opponent for training and evaluation. Customizable difficulty levels.

⚡

Headless Mode

Train agents at high speed without rendering overhead. Full game logic works independently of UI for maximum performance.

Quick Start

Get up and running in minutes with standard Gymnasium interface

# Install dependencies
pip install pygame numpy gymnasium stable-baselines3

# Import the environment
from rl.gym_env import StrategyGameEnv

# Create environment
env = StrategyGameEnv(
    opponent='bot',
    render_mode=None
)

# Standard Gym API
obs, info = env.reset()
action = env.action_space.sample()
obs, reward, terminated, truncated, info = env.step(action)

# Train with Stable-Baselines3
from stable_baselines3 import PPO

model = PPO('MultiInputPolicy', env, verbose=1)
model.learn(total_timesteps=100000)
model.save('ppo_tactics')

Installation

Basic Installation (GUI Mode)

pip install pygame pandas numpy

Full Installation (with RL)

pip install pygame pandas numpy gymnasium stable-baselines3[extra]

Play the Game

python main.py --mode play

Train an Agent

python main.py --mode train --algorithm ppo --timesteps 100000
